# Documentation

This directory follows the [Diataxis](https://diataxis.fr) framework.
Every document belongs to exactly one of four types.

| Type | Location | Answers |
|---|---|---|
| **Tutorial** | [tutorials/](tutorials/) | "How do I learn this?" |
| **How-to guide** | [how-to-guides/](how-to-guides/) | "How do I achieve X?" |
| **Explanation** | [explanation/](explanation/) | "Why does this work this way?" |
| **Reference** | [reference/](reference/) | "What exactly does X do/accept?" |
